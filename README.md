ArtNexus Gallery
    Página web estática diseñada para una galería de arte contemporáneo, con enfoque en diseño limpio, secciones amplias y experiencia visual

    Este proyecto es una landing page moderna para ArtNexus Gallery, compuesta por tres secciones principales:
        1. Inicio: Mensaje de bienvenida con con un hiro banner centrado
        2. Servicios: Tarjetas informativas sobre los servicios de la galería
        3. Galería: Grid de imágenes con obras contemporáneas
    Incluye un header fijo, menús con animaciones, diseño responsivo y estilos minimalistas

Tecnologías utilizadas
    1. HTML5
    2. CSS3
    3. Diseño responisve (media queries)
    4. Tipografias estándar del navegador

Características principales
    1. Header fijo: Un menú que permanece visible al hacer scroll, con animacion de subrayado suave al pasar el curso
    2. Seccion Hero: Gran área de bienvenida con tipografia elegante y mensajes introductorios
    3. Galería: Diseñada con CSS Grid, mostrando imágenes en columnas adaptadas según el tamaño de pantalla
    4. Servicios: Tarjetas con sombra y borde redondeado que se acomodan de forma flexible en tablets y móviles
    5. Responsive: Incluye breakpoint para:
        1024px - 768px
        Menos de 768px
    
    Cada vista reorganiza el contenido para mantener legibilidad y estética en pantallas reducidas

Como usar el proyecto
    1. Clona o descarga este repositorio: git clone https://github.com/usuario/artnexus-gallery.git
    2. Asegúrate de tener la carpeta public/con las imagenes dentro del proyecto
    3. Abre el archivo index.html en tu navegador
    4. ¡Listo! Ya puedes visualizar la página

Archivos principales
    index.html: Contiene la estructura del sitio: header, secciones, imágenes y footer.

    styles.css, Incluye:
        Estilos principales
        Animaciones del menú
        Diseño de tarjetas
        Grid de galería
        Media queries responsive
